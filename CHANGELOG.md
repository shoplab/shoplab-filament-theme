# Changelog

All notable changes to `shoplab-filament-theme` will be documented in this file.

## v1.0.0 - 2026-01-22

### Added
- Initial release of Shoplab Filament Theme
- Linear-inspired design system for Filament v5
- Custom CSS tokens for colors, typography, and spacing
- Support for light and dark modes
- Plugin-based integration with Filament panels
- Support for viteTheme() compilation method
- Comprehensive documentation and usage examples
- GitHub Actions CI/CD pipeline
- Pest test suite
- Laravel Pint code style configuration

### Features
- Zero JavaScript - pure CSS theming
- Automatic service provider registration
- Flexible integration options (plugin or viteTheme)
- Optimized for multi-tenant applications
- Built with Tailwind CSS 4

### Requirements
- PHP 8.2+
- Laravel 11+
- Filament 5.0+
